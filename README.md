# 使用方法
	pull代码到本地
	npm install安装依赖
	npm run build
	根目录lib文件夹中即为build完成的代码
	import { Radar } from 'xx/xx'

# 目前已完成
	line（折线图）
	line-dot（散点图）
	radar（雷达图）
	dash-board（仪表盘）
	clock（时钟）
